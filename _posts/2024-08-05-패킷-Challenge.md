# Wireshark를 사용하여 Naver접속 시 사용된 패킷 필터링하기

```sh
# 입력
ping naver.com
```

```sh
# 결과
Ping naver.com [223.130.192.247] 32바이트 데이터 사용:
```

> 네이버 IP 확인

|

1. "ip.addr == 223.130.192.247"로 패킷 필터링

2. 네이버 접속 후 트래픽 발생

3. Stop

4. 패킷 분석
